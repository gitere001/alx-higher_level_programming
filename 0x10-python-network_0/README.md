## Web Development Learning Resources

### Overview

Welcome to the **Web Development Learning Resources** directory! This repository contains various files and materials to help you understand fundamental concepts of web development, including URLs, HTTP, cURL, and more. Whether you're a beginner or looking to refresh your knowledge, you'll find valuable resources here.

### Contents

- **URL Basics**: Understand the structure and components of URLs.
- **HTTP Fundamentals**: Learn about the Hypertext Transfer Protocol, including requests, responses, status codes, and headers.
- **cURL Usage**: Practice making HTTP requests from the command line using cURL.

### How to Use

1. **Clone the Repository**: Start by cloning this repository to your local machine:

    ```
    git clone <repository_url>
    ```

2. **Navigate to the Directory**: Enter the directory containing the learning resources:

    ```
    cd 0x10-python-network_0
    ```

3. **Explore the Materials**: Review the files in each category to enhance your understanding of web development concepts.

    - `url_basics.md`: Read this markdown file to grasp the fundamentals of URLs.
    - `http_fundamentals.pdf`: Explore this PDF document for a comprehensive overview of HTTP basics.
    - `curl_examples.sh`: Execute this shell script to practice making HTTP requests with cURL.
